Patterns Store Front is a modern, block-based WordPress theme designed to integrate seamlessly with the Patterns Store plugin.

== Description ==

Patterns Store Front is a specialized theme built to work alongside the Patterns Store plugin. This theme provides a streamlined way to showcase pattern kits, individual patterns, and theme JSON files within a beautiful and responsive block-based environment. With full support for WordPress Full Site Editing (FSE), Patterns Store Front allows users to customize headers, footers, layouts, and global styles effortlessly, all within the WordPress Site Editor.

The theme comes with built-in templates to display patterns, pattern kits, and previews, making it the perfect companion for designers and developers looking to manage and present design elements in an organized and professional manner.

Whether you’re showcasing a personal collection of patterns or running a public repository of pattern kits, Patterns Store Front ensures that your designs are displayed in a visually engaging and user-friendly format.

== Frequently Asked Questions ==

= How to install theme? =

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
